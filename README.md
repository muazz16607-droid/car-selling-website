# Car Marketplace Website

A responsive car-selling frontend built with HTML, CSS, and JavaScript.

## Features
- Browse car listings
- View car details
- Admin-only car posting
- Admin car deletion
- Contact page with social links
- Animated UI and hover effects

## Run Locally
Open `index.html` in your browser  
or use VS Code Live Server.

## Project Structure
- `index.html` - Home page
- `cars.html` - Listings page
- `car.html` - Car details
- `sell.html` - Add listing (admin only)
- `admin.html` - Admin dashboard
- `contact.html` - Contact page
- `assets/styles.css` - Styling
- `assets/app.js` - App logic

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Note
This project uses browser `localStorage` and demo authentication (not production-secure).
